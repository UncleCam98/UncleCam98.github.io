# UncleCam98.github.io

Statisk portfolio byggd med HTML, CSS och JavaScript.

Startsidan i roten anvander samma struktur som `website-sample`, men ar nu uppsatt som en fristaende portfolio med:

- hero-sektion
- om-sektion
- filtrerbara kompetenser
- projektkort
- erfarenhetssektion
- kontaktsektion

Det snabbaste sattet att anpassa sidan ar att uppdatera:

- `sections/*.js` for texter och rubriker
- `data/*.js` for projekt och kompetenser
- `styles.css` for farger och uttryck
- `assets/profile.jpg` om du vill visa ett portratt
